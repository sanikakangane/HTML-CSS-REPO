# 🛍️ ShopKart E-Commerce Website

> A bright and modern **product showcase website** built using **HTML & CSS**, designed to display tools, safety equipment, and electrical products in a clean card layout. ✨

---

## 🚀 Features

- 🛍️ **Product Card Layout** with pricing and discounts  
- 🛠️ **Safety Equipment Section**  
- ⚡ **Electrical Products Section**  
- 💸 **Price, Discount, and Old Price Display**  
- 🎨 **Bright Gradient UI Design**  
- ✨ **Hover Effects on Product Cards**  
- 📱 **Responsive Grid Layout**  

💡 **Tip:** Perfect for beginners learning how to build an **e-commerce product UI using HTML & CSS**.
